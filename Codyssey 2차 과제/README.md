## 배달모아 브랜드 아이덴티티

- 1. 브랜드 핵심 정의
    
    > **혼자서는 부담스러운 배달 주문을 가까운 사람들과 모아, 먹고 싶은 메뉴를 더 가볍게 즐길 수 있도록 돕는 공동배달 플랫폼**
    > 
- 2. 타깃
    
    **핵심 타깃**
    
    - 대학생 및 기숙사 거주자
    - 원룸촌에 거주하는 20~30대 자취생
    - 1인분만 주문하고 싶지만 최소 주문금액과 배달비가 부담되는 사용자
    - 배달앱 사용에는 익숙하지만 주변에서 같이 주문할 사람을 찾기 어려운 사용자
    
    **확장 타깃**
    
    - 점심이나 야근 시간에 함께 주문하는 직장인
    - 오피스텔·아파트 등 공용 수령 거점을 이용할 수 있는 1인 가구
- 3. 브랜드 톤앤매너
    
    따뜻한 웜톤 / 현실적인 일상 공간 / 잔잔한 시네마틱 분위기 / 깔끔한 디지털 UI / 어두움에서 밝음으로의 전환 / 밝고 활기찬 마무리 
    
- 4. 핵심 차별점(USP)
    
    #### 최소 주문금액 장벽 완화
    
    혼자서는 최소 주문금액을 채우기 어려웠던 식당도 여러 사용자의 주문을 합쳐 이용할 수 있다. 불필요하게 사이드 메뉴를 추가하지 않고 **실제로 먹고 싶은 메뉴만 주문할 수 있다.**
    
    #### 배달비 공동 부담
    
    하나의 배달비를 여러 참여자가 나눠 부담해, 최근 높아진 배달비에 대한 부담을 줄인다.
    
    #### 주변 사용자 자동 연결
    
    사용자가 단체 채팅방이나 커뮤니티에서 직접 사람을 모집하지 않아도, 위치와 주문 시간 등이 맞는 공동주문 파티를 확인하거나 추천받을 수 있다.
    
    #### 공유 장바구니
    
    여러 사람이 함께 주문하더라도 참여자별 메뉴와 금액을 명확하게 구분한다. 누가 어떤 메뉴를 주문했는지 헷갈리는 문제를 줄일 수 있다.
    
    #### 자동 정산
    
    음식값과 배달비를 자동으로 계산하고 결제 완료 여부를 표시한다. 개인 계좌이체나 수동 계산 과정에서 발생하는 불편과 금전 분쟁을 줄인다.
    
    #### 안전한 공동 수령과 신뢰 관리
    
    기숙사 로비·학교 건물 입구·아파트 공용 공간 등 공개된 거점에서 음식을 수령하고, 리뷰와 매너 점수를 통해 반복적인 노쇼와 지각을 관리한다.
    
- 5.  핵심 메시지
    
    “주문은 모으고, 배달비는 나누고.”
